<h1>游戏规则</h1>
<p>
	* 玩家通过键盘上的上下左右键进行移动<br>
	* 玩家无法移动至屏幕外<br>
	* 敌人会在屏幕上从左至右移动，并在到达右端后随机选择一行继续移动<br>
	* 玩家撞到敌人后会自动退回到最下面两行随机位置<br>
	* 玩家在到达小河上并获取到过关钥匙即为闯关成功<br>
	* 玩家闯关成功后会出现恭喜动画，显示该局游戏完成时间，并选择重试来进行下一局游戏<br>
</p>
